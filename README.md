# 💳 Bank Transaction Analytics Dashboard

An interactive Banking Analytics Dashboard built with **HTML, CSS, JavaScript, and Chart.js** that transforms transaction data into actionable business insights through dynamic visualizations, real-time KPIs, and Power BI-inspired cross-filtering interactions.

---

## 🚀 Features

- 📊 Interactive and responsive dashboard
- 🔄 Cross-filtering across all visualizations
- 📈 Dynamic KPI cards with real-time updates
- 🎯 Multi-level filtering (City, Gender, Month, Balance Tier, Amount Range)
- 📉 Advanced charts powered by Chart.js
- 📱 Fully responsive design
- ⚡ Fast client-side performance
- 🎨 Modern UI with professional dashboard layout

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- AI-Assisted Development (Claude & ChatGPT)

---

## 📊 Dashboard Highlights

- Monthly Transaction Trends

- Transaction Amount Distribution
- Interactive KPI Metrics
